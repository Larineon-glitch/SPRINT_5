# Sprint_5 — Автотесты для Stellar Burgers

Этот проект содержит автоматизированные тесты для веб-приложения "Stellar Burgers".
Сайт: 
[stellarburgers.education-services.ru]
(https://stellarburgers.education-services.ru/)


# Технологии
- Python 3.9
- Selenium WebDriver — для управления браузером
- Pytest — для запуска и организации тестов
- ChromeDriver — для работы с Google Chrome


# Что проверяется
# 1. Регистрация
- Успешная регистрация с валидными данными
- Ошибка при вводе пароля короче 6 символов

# 2. Вход в аккаунт
- Через кнопку «Войти в аккаунт» на главной странице
- Через кнопку «Личный кабинет»
- Через кнопку «Войти» в форме регистрации
- Через кнопку «Войти» в форме восстановления пароля

# 3. Личный кабинет
- Переход в личный кабинет по клику на «Личный кабинет»
- Переход из личного кабинета в конструктор по кнопке «Конструктор»
- Переход из личного кабинета на главную по клику на логотип

# 4. Конструктор
- Переход к разделу «Булки»
- Переход к разделу «Соусы»
- Переход к разделу «Начинки»

# 5. Выход из аккаунта
- Выход по кнопке «Выйти» в личном кабинете
